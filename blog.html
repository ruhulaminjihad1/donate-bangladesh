What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree of objects that can be manipulated with code, typically JavaScript. Each element in the HTML of the web page, such as headings, paragraphs, links, and images, is represented as a node in this tree
